# Granjacema

Site estático de apresentação da Granjacema — pequena granja local com produção de ovos.

## Sobre

Este é um site estático simples (HTML/CSS) que apresenta a história, unidades e linha de produtos da Granjacema.

## Estrutura

- `index.html` — página principal do site.
- `style.css` — folhas de estilo.
- `img/` — imagens usadas no site.

## Como executar

Basta abrir `index.html` em um navegador (duplo clique ou arrastar para a janela do navegador).

## Personalizações

- Para atualizar o link "Saber mais" no cabeçalho que aponta para o repositório GitHub, edite o elemento de navegação em `index.html` e substitua `https://github.com/SEU_USUARIO/REPO` pela URL do seu repositório.
